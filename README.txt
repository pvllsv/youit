Использование:
1) ввести колличество ожидаемых строк
2) последовательно ввести  проверяемые строки

Основная логика:
1)проверяем побуквенно патерн youit (увеличиваем результат за каждое совпадение)
2)если имвол несоответствует, проверяем не являеться ли этот символ началом новой последовательности
3) если являеться сохраняем прошлый результат и начинам подсчет нового с 1
4) если не являеться сохраняем результат и начинаем посчет с 0
5) из всех результатов находим самый максимальный

результаты тестов:
youityou-8
bbbbbybbb-1
t-0
y-1
youitxyouityouit-10
yoyoyoy-2
